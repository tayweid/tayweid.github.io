The first part of the class starts by developing the core skills in exploratory data analysis and visualization, done primarily in Excel.

Classwork

Part 0 // Welcome (Slides )

Excel Exercise: Installation

This exercise walks through the steps to install Excel on your computer.

Part 1.1 // Visualizing Categorical Data (Slides )

Excel Exercise ( Data | Example )

This exercise uses a dataset on the locations of a small coffee shop chain to visualize a cateogrical variable.

Part 1.2 // Visualizing Continuous Data (Slides )

Excel Exercise ( Data | Source | Example )

This exercise uses a dataset on the profiles of Starbucks customers to visualize a cateogrical variable.

Part 1.3 // Comparing Continuous Variables by Cateogry (Slides )

Excel Exercise ( Data | Source | Example )

This exercise uses a dataset on the transactions of a small chain of coffee shops to visualize a continuous variable within cateogries.

Part 1.4 // Filtering Data (Slides )

Excel Exercise ( Data | Source | Example )

This exercise uses a dataset on the hours of Starbucks locations globally to better understand a few subsets of the data by filtering.

Part 1.5 // Transforming Data (Slides )

Excel Exercise: Transformations ( Data )

This exercise uses a dataset on Starbucks locations' hours of operation.

Part 1.6 // Exploring Relationships (Slides )

Excel Exercise: Scatterplots ( Data )

This exercise uses a scatterplot to explore the relationship between two variables in a dataset on the GDP and coffee production of some coffee producing countries.

Excel Exercise: Boxplots ( Data )

This exercise uses a boxplot to explore the distribution of coffee consumption over time in a dataset on the coffee consumption per capita.

Part 1.7 // Data Connected Through Time (Slides )

Excel Exercise: Timeseries ( Data | Source | Example )

This exercise uses coffee price data to build a timeseries plot of coffee prices in today's dollars.
