# ECON 0150 F26 Proposed Weekly Schedule

Assumes ~2 class sessions per week, 15-week semester.

## Week-by-Week

| Week | Block | Topic | Key Move |
|------|-------|-------|----------|
| 1 | **0** | Framework | Classify data, x_it notation, all structures named |
| 2 | **1.1** | Numerical Variables | Histograms (i-indexed), line plots (t-indexed), mean, SD |
| 2-3 | **1.2** | Populations and Samples | "Your histogram is one sample. What about the truth?" |
| 3 | **1.3** | Central Limit Theorem | CLT, standard error, dice simulation |
| 3-4 | **1.4** | CIs & Hypothesis Testing | t-distribution, CIs, p-values |
| 4 | **1.5** | The Simplest GLM | y ~ 1, MSE, testing the intercept — Part 1 capstone |
| | | **MiniExam 1** | |
| 5 | **2.1** | Categorical Variables | Bar charts, frequency tables, Python: value_counts(), countplot() |
| 5-6 | **2.2** | Numerical × Categorical | Grouped boxplots, log transform, BOGO question |
| 6 | **2.3** | Comparing Groups | Dummy variables, t-test as regression — "Is the difference real?" |
| 6-7 | **2.4** | Panel Data & Pre/Post | Card & Krueger payoff, long/wide format, comparing changes across groups |
| 7-8 | **2.5** | Scatterplots & Regression | Scatterplots → OLS, slope testing |
| 8-9 | **2.6** | Model Diagnostics | Four assumptions, autocorrelation as independence case study |
| | | **MiniExam 2** | |
| 10 | **3.1** | Conditional Relationships | Elections: income vs voting flips by category |
| 10-11 | **3.2** | Categorical Controls | Simpson's Paradox, fixed effects, wage gap |
| 11-12 | **3.3** | Interactions | Different slopes by group |
| 12-13 | **3.4** | Numerical Controls | Multiple predictors, "holding constant" |
| | | **MiniExam 3** | |
| 14 | **4.1** | Analysis Workflow | Question → data → model → interpretation |
| 14-15 | **4.2** | Presenting Findings | Slides, narrative, examples |
| Finals | | **Final Project Presentations** | |
