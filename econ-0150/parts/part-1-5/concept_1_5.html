Counting Changes

How many countries increased vs decreased?


We can see visually that most points are above the 45° line.

But how do we count exactly how many increased or decreased?

> we can create a change column and then filter

Counting Changes: Create a Column

First, compute the change for each country 

# Create a change column
percap['change'] = percap['2019'] - percap['1999']
percap.head()
Code 1999 2019 change
AUT 8.43 7.93 -0.50
BGR 2.65 3.64 0.99
HRV 4.48 5.62 1.14
… … … …

Counting Changes: Filter

Then, filter to count increases and decreases 

# Count countries that increased (change > 0)
increased = percap[percap['change'] > 0]
len(increased)
# Count countries that decreased (change < 0)
decreased = percap[percap['change'] < 0]
len(decreased)

> filtering uses square brackets with a condition inside

A New Question

How has each country’s consumption changed over time?



We’ve compared distributions across years and tracked changes between 1999 and 2019.

But what if we want to see each country’s full trend over all 30 years?

> we need a line plot with Year on x-axis and Consumption on y-axis, colored by Country

The Problem

Wide format doesn’t work for line plots


In wide format, years are column names, not values in a column.

# This won't work — there's no 'Year' column!
sns.lineplot(percap, x='Year', y='Consumption', hue='Code')

> we need to reshape from wide to long format

Reshaping: Wide → Long

Use melt() to turn columns into rows

> each year column becomes rows in a new “Year” column

Reshaping: The Code

Use melt() to reshape wide → long 

# Wide to Long
percap_long = percap.melt(
    id_vars=['Code'],        # Keep as identifier columns
    var_name='Year',         # Name for the former column headers
    value_name='Consumption' # Name for the values
)

> id_vars = columns to keep, everything else gets melted

Exercise 1.5 | Reshaping

How has each country’s consumption changed over time? 

# Reshape wide to long
percap_long = percap.melt(
    id_vars=['Code'],
    var_name='Year',
    value_name='Consumption'
)
# Now we can plot trends for each country
sns.lineplot(percap_long, x='Year', y='Consumption', hue='Code')

Reshaping: Long → Wide

Use pivot() to turn rows into columns


Sometimes data comes in long format but we need wide format for boxplots or scatterplots.

pivot() is the opposite of melt():

  • melt() turns columns into rows (wide → long)
  • pivot() turns rows into columns (long → wide)

Reshaping: The Code

Use pivot() to reshape long → wide 

# Long to Wide
percap_wide = percap_long.pivot(
    index='Code',       # What identifies each row
    columns='Year',     # What becomes column headers
    values='Consumption' # What fills the cells
)

> index = row identifier, columns = new column names, values = cell values

Two Directions of Reshaping

Choose based on your starting point and goal


Direction Function When to use
Wide → Long melt() Need line plots, faceting
Long → Wide pivot() Need boxplots, scatterplots

> the data format should match your visualization goal

Summary

  • Wide format: Each time period is a column
  • Multi-boxplots compare distributions across time
  • Scatterplots with 45° lines track individual changes
  • Filtering with df[df['col'] > 0] counts subsets
  • melt() converts wide → long when you need line plots
  • pivot() converts long → wide when you need boxplots or scatterplots

Long vs Wide: The Complete Picture

Choose format based on your task


Long Format (Part 1.4)

  • Comparing groups at a point in time
  • Faceting, line graphs, groupby
  • hue='Group', col='Group'

Wide Format (Part 1.5)

  • Comparing time periods for the same units
  • Boxplots, scatterplots, correlations
  • Each column is a time period
